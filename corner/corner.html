Silicene

Silicene is a open source messanger in browser. Secure. Transparent. Simple.

Open source features

Secure

All messages are encrypted in browser. No one can read your messages.

Transparent

All messages are transparent. You can see all messages in one place.

Simple

Simple and easy to use. You can use it without any knowledge of programming.
